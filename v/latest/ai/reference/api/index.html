Section titled “RAG”

ExportWhat it is
RagReturned by ai.rag. .index(), .retrieve(), .clear(), .asTool(), .name.

RetrieveOptions{ topK?, threshold?, candidates?, tags? }.

VectorStoreVector-store adapter. cacheVectorStore wraps a @warlock.js/cache driver.
